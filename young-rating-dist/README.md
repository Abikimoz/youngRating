# Инструкция по развертыванию на сервере без интернета

Это руководство по запуску приложения "YoungRating" на сервере, который не имеет доступа к сети Интернет.

## 1. Сборка архива с образами (выполняется на машине с доступом в интернет)

Перед копированием файлов на сервер необходимо собрать `images.tar`, который будет содержать все необходимые Docker-образы.

1.  **Соберите образы frontend и backend**
    Убедитесь, что у вас установлены `docker` и `docker-compose`. В корневой папке проекта выполните:
    ```bash
    docker compose -f docker/docker-compose.yml build --no-cache
    ```

2.  **Загрузите образ PostgreSQL**
    ```bash
    docker pull postgres:15-alpine
    ```

3.  **Создайте `images.tar`**
    Сохраните все три образа в один архив:
    ```bash
    docker save -o young-rating-dist/images.tar young-rating-backend:latest young-rating-frontend:latest postgres:15-alpine
    ```

## 2. Процесс запуска на сервере без интернета

### Предварительные требования

На сервере должны быть установлены:
- `Docker`
- `Docker Compose`

### Процесс запуска

1.  **Скопируйте файлы**
    Убедитесь, что в одной директории на сервере находятся два файла:
    - `images.tar` (архив с Docker-образами, созданный на предыдущем шаге)
    - `docker-compose.yml` (файл конфигурации для запуска)

2.  **Зайдите в директорию**
    Перейдите в директорию с этими файлами.

3.  **Выполните запуск**
    Выполните последовательно две команды:

    ```bash
    # 1. Загрузка образов в локальное хранилище Docker
    docker load -i images.tar

    # 2. Запуск приложения в фоновом режиме
    docker compose up -d
    ```

## 3. Проверка и управление

- **Проверить статус контейнеров:**
  ```bash
  docker compose ps
  ```

- **Посмотреть логи приложения:**
  ```bash
  docker compose logs -f
  ```

- **Остановить приложение:**
  ```bash
  docker compose down
  ```
